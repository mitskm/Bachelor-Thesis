This is a repository for Bachelor Thesis “(IL)LIBERALISM: EFFECTS OF MULTIDIMENSIONAL IDEOLOGICAL POSITIONS ON SUPPORT FOR
POLITICAL CANDIDATES”

It contains files that are necessary for the replication of this work.

Questionnaire materials:

1. “questionnaire_full.pdf” – full PDF of the questionnaire used in this study, downloaded from 1ka.si.
2. “conjoint_vignettes_for_subm.txt” – txt document containing all conjoint pairs used in this study.
3. “vignette_generation_and_randomization_check.ipynb” – full ipynb used for automatically generating conjoint pairs and conducting randomization checks.
4. “full_survey_upd.html” – raw HTML code of the survey editor. This file was used for randomization and administration checks.

Datasets:

1. “anketa511686-2026-04-16.xlsx” – raw xlsx file downloaded from 1ka.si containing survey responses.
2. “conjoint_df.csv” – long-format dataset with all necessary covariates. This dataset is used for the main analysis.

Replication Files:

1. “Replication_File_Data_Handling.ipynb” – Python code for replicating the creation of “conjoint_df.csv”. Includes code for data preprocessing and descriptive results.
2. “Replication_File_Data_Analysis.R” – R script for replicating all main analyses.
3. "power_analysis_thesis.R" – R script for replicating the power analysis. 

Miscellaneous:

1. “example_tasks_for_cognitive_interview.txt” – txt file containing example tasks used for cognitive interviews.
